A backbone-webpack example
===============

[![Build Status][travis-image]][travis-url]

Install
-----

    $ npm install

Development
-----

    $ npm run d

Go to http://localhost:3000 to launch your app.

Public
-----

    $ npm run p

Productions:
- public/: chunks
- index.jsp: entry
- webpack-assets.json: hash of chunks
